# xetead-bot

TRPG 캠페인의 전투를 자동 정산하는 Discord 봇.

---

## 전투 시스템

### 전장

전장은 **7열 × 진영(아군/적군)** 구조다. 각 열에는 최대 3명이 위치할 수 있다.

```
적군  [1열] [2열] [3열] [4열] [5열] [6열] [7열]
아군  [1열] [2열] [3열] [4열] [5열] [6열] [7열]
```

### 캐릭터 스탯

| 스탯    | 설명                                           |
|-------|----------------------------------------------|
| 공격력   | 기본 공격 및 공격 계수 스킬의 주사위 상한                     |
| 체력    | 최대/현재 HP                                     |
| 사거리   | 공격 가능한 열 거리 (1 = 인접, 3 = 원거리)                |
| 공격 속성 | 물리 / 마법                                      |
| 마법 저항 | 낮음(+10%) / 보통(±0) / 높음(−10%). 마법 공격을 받을 때 적용 |
| 코스트   | 턴당 행동 자원. 기본 3                               |

### 라운드 페이즈

한 라운드는 4개의 페이즈로 진행된다.

```
1. 적 행동 선언   ─ 적군 커맨드 입력. 이동과 PRE 버프 즉시 처리, 공격은 예약.
2. 아군 행동      ─ 아군 커맨드 입력. 모든 효과 즉시 정산.
3. 적 공격 정산   ─ 예약된 적군 공격 대미지/POST 버프 정산.
4. 라운드 종료    ─ 버프/디버프 턴수 차감, 만료 버프 제거.
```

### 커맨드 형식

커맨드는 `[행동1 - 행동2 - ...]` 형태로 작성하며, `-`로 복수 행동을 연결한다.

| 형식          | 예시            | 설명                 |
|-------------|---------------|--------------------|
| `이동/열`      | `이동/3`        | 지정 열로 이동           |
| `공격/대상이름`   | `공격/에너미`      | 기본 공격              |
| `스킬/스킬명/대상` | `스킬/불꽃구슬/에너미` | 이름 지정 스킬           |
| `스킬/스킬명/열`  | `스킬/광역폭발/2`   | 열 지정 스킬            |
| `스킬/스킬명`    | `스킬/자기강화`     | 대상 없는 스킬 (자기 자신 등) |

예시: `/행동 이름:테스트 커맨드:"[이동/2 - 공격/에너미]"`

### 검증 항목

커맨드 실행 전 아래 항목을 순서대로 검증하며, 하나라도 실패하면 전체 커맨드가 취소된다.

1. 커맨드 사용자가 전장에 존재하는지
2. 지정한 스킬이 캐릭터에 등록되어 있는지
3. 스킬 대상 수가 `target_count`를 초과하지 않는지
4. 커맨드 전체 코스트가 잔여 코스트 이내인지
5. 공격/힐 대상이 전장에 존재하는지, 사거리 내인지
6. 이동 목적지에 자리가 남아있는지

### 대미지 계산

```
최종 대미지 = floor(기본값 × (1 + Σ배율 보정)) + Σ정수 보정
```

- **기본 공격**: `nd6(마일스톤N, 공격력)` 주사위 굴림 결과
- **고정값 스킬**: 스킬에 명시된 수치
- **계수 스킬**: `nd6(마일스톤N, 공격력) × 계수`
- 마법 공격자가 마법 저항 대상을 공격하면 저항 배율이 추가 적용된다

### 버프/디버프 시스템

버프는 **지속 시간** 방식에 따라 두 종류다.

- **턴수제**: 라운드 종료 시 차감. 0이 되면 제거.
- **횟수제**: 공격 시 또는 피격 시 차감.
- **패시브**: 지속 시간 없음, 영구 유지.

**발동 타이밍**

| 타이밍              | 발동 시점    |
|------------------|----------|
| `ON_ROUND_START` | 라운드 시작 시 |
| `ON_ACTION`      | 공격/피격 시  |
| `ON_ROUND_END`   | 라운드 종료 시 |

**구현된 버프 목록**

| 버프 클래스               | 효과                      |
|----------------------|-------------------------|
| `BuffAtk`            | 공격력 수치 증가               |
| `BuffReceivedDamage` | 받는 대미지 배율 변경            |
| `BuffGivenDamage`    | 주는 대미지 배율 변경            |
| `BuffDamageOverTime` | 매 라운드 지속 대미지            |
| `BuffHealOverTime`   | 매 라운드 지속 회복             |
| `BuffNoDamage`       | 대미지 무효                  |
| `BuffNoHeal`         | 회복 무효                   |
| `BuffTaunt`          | 도발 — 홀더의 모든 공격이 도발자를 향함 |

### 스킬 대상 규칙

| 규칙 클래스                  | 대상 지정 방식          |
|-------------------------|-------------------|
| `SkillTargetRuleSelf`   | 항상 자기 자신 (입력 무시)  |
| `SkillTargetRuleNamed`  | 이름으로 지정한 캐릭터      |
| `SkillTargetRuleColumn` | 열 번호로 지정, 해당 열 전원 |

## Discord 슬래시 커맨드

| 커맨드     | 설명               |
|---------|------------------|
| `/전투`   | 새 전투 세션 시작       |
| `/캐릭터`  | 아군 추가            |
| `/적군`   | 적군 추가            |
| `/스킬`   | 캐릭터에 스킬/패시브 지정   |
| `/스킬목록` | 사용 가능한 스킬 조회     |
| `/버프목록` | 사용 가능한 패시브 버프 조회 |
| `/시작`   | 전투 시작 (페이즈 진입)   |
| `/행동`   | 캐릭터 커맨드 입력       |
| `/페이즈`  | 다음 페이즈로 전환       |
| `/현황`   | 현재 전장 상태 표시      |
| `/종료`   | 세션 종료            |

세션은 채널당 하나이며, 장시간 미활동 시 자동 종료된다.

## 실행

```bash
# 의존성 설치
pip install -r requirements.txt

# 환경 변수 설정 (.env)
DISCORD_BOT_TOKEN=...
GOOGLE_SPREADSHEET_CREDENTIALS='{...}'
DB_SPREADSHEET_KEY=...

# 봇 실행
python app/bots/discord_bot/main.py
```
